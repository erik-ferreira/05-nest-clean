# Testes

- Por padrão toda a compilação de typescript para javascript é feita pela cli do typescript no NestJS

## Configs SWC + Vitest

- No package.json é adicionado o seguinte comando:

  ```json
  "scripts": {
    // Nele os testes e2e são testados em cima desse arquivo específico
    "test:e2e": "vitest run --config ./vitest.config.e2e.ts"
  }
  ```

- No arquivo `vitest.config.e2e.ts`:
  ```ts
  test: {
    // Esse arquivo vai ser executado antes de cada arquivo de teste e2e
    setupFiles: ["./test/setup-e2e.ts"]
  }
  ```
